mod1.7z

    FS1 = File Size from mod1
    FS2 = File Size from mod2

    NTR = Normalized modification Time Ratio of overlapping files
    FCR = File Count Ratio of all files
    Score = NS * FCR
Values based on the whole mod:
    FC = File Count
    Size = Total (uncompressed) Size of the mod
